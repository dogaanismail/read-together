# Security Policy

## Supported Versions
The latest main branch is supported with security updates.

## Reporting a Vulnerability
Please report security issues privately:

- Open a private security advisory on GitHub if available, or
- Email the maintainer (see https://github.com/dogaanismail profile for contact)

We’ll acknowledge receipt within 72 hours and work on a fix or mitigation plan. Please do not disclose publicly until a fix is released.
# Code of Conduct

We follow the Contributor Covenant Code of Conduct, version 2.1.

- Homepage: https://www.contributor-covenant.org
- Full text: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

In short: be kind, inclusive, and respectful. We’re building a safe space for people who stutter. Harassment or discrimination is not tolerated.

## Reporting
Report unacceptable behavior privately via GitHub Discussions (private) or email the maintainer listed on the project profile. We will review and act as promptly as possible.
